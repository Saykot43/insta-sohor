How does JavaScript Work?
Answer :

JavaScript code runs in the browser JS Engine. JavaScript is what is called a Client-side Scripting Language. That means that it is a computer programming language that runs inside an Internet browser (a browser is also known as a Web client because it connects to a Web server to download pages). When the browser loads the page, the browser has a built-in interpreter that reads the JavaScript code it finds on the page and runs it.

How Event loop Works?
Answer :

Event Loop is the secret behind asynchronous programming of JavaScript. JavaScript runs all operations on a single thread, but using a few smart data structures, it gives us multi-threading. It flow the rules (even loop->add a new operation if stack is empty->call stack->call to api->browser apis->send to event queue after timeout->event queue->event loop)
